# Фронтенд для LMS системы

## Структура

### HTML страницы
- `form.html` - Страница авторизации и регистрации
- `lkp.html` - Личный кабинет преподавателя
- `lks.html` - Личный кабинет студента (заглушка)
- `course.html` - Страница управления курсом

### JavaScript
- `scripts/api.js` - Класс для работы с API (запросы, авторизация)
- `scripts/form.js` - Логика авторизации/регистрации
- `scripts/prepod.js` - Логика ЛК преподавателя (CRUD курсов)
- `scripts/course.js` - Логика страницы курса (материалы, задания, проверка работ, статистика)
### CSS
- `styles/reset.css` - Сброс стилей браузера
- `styles/form.css` - Стили формы авторизации
- `styles/lk.css` - Стили личного кабинета
- `styles/course.css` - Стили страницы курса и модальных окон

### Иконки
- `img/psb.png` - Логотип банка
- `img/*.svg` - Различные иконки на сайте

## Функционал преподавателя

### Курсы
- Просмотр списка курсов
- Создание нового курса
- Редактирование курса
- Удаление курса
- Переход к детальной странице курса

### Материалы курса
- Просмотр материалов
- Добавление материала
- Редактирование материала
- Удаление материала

### Задания курса
- Просмотр заданий
- Создание задания с дедлайном и максимальными баллами
- Редактирование задания
- Удаление задания
- Просмотр сданных работ студентов

### Проверка работ
- Просмотр работ студентов по заданию
- Выставление оценок и комментариев
- Обновление оценок

### Статистика
- Общая статистика по курсу
- Прогресс каждого студента
- Средние баллы

## Стек
- Vanilla HTML5
- Vanilla CSS3
- Vanilla JavaScript (ES6+)
- Fetch API для запросов
